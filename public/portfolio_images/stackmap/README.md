# Stack Map image slots

Optional hero/general images:

- SM-01 `/portfolio_images/stackmap/stackmap-hero.jpg`
- SM-02 `/portfolio_images/stackmap/systems-diagram.jpg`
- SM-03 `/portfolio_images/stackmap/workstation-stack.jpg`

Optional per-skill images use this pattern:

`/portfolio_images/stackmap/<skill-slug>.jpg`

Examples:

- `/portfolio_images/stackmap/python.jpg`
- `/portfolio_images/stackmap/computer-vision.jpg`
- `/portfolio_images/stackmap/sensor-fusion.jpg`
- `/portfolio_images/stackmap/microsoft-graph-api.jpg`
