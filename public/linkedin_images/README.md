# LinkedIn / event image pipeline

Manually download useful images from LinkedIn, events, demos, or project screenshots and place them in this folder.

Recommended filenames:

- `starkhacks-team-2026.jpg`
- `ecocar-team.jpg`
- `career-bootcamp.jpg`
- `networking-event-ink-factory.jpg`
- `robotics-workshop.jpg`
- `igda-gamebites.jpg`
- `observ-e-demo.jpg`

Then add metadata in `src/data/linkedinImages.ts` or reference images directly inside the relevant project file under `src/content/projectPages/`.
